To open and test the extension:
1. Go to chrome://extensions/
2. Enable Developer mode
3. Unzip the compressed .zip folder
4. In chrome://extensions/ click 'Load Unpacked'
5. Select the unzipped extension folder

If you have any questions, do not hesitate to ask and I am happy to help.

Thank you for ordering! :)
